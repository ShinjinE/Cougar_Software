Manual Control¶

This module will handle all input from the Bluetooth connected PS4 controllers in an asynchronous manner. It expects two controllers to be connected and will not run until then. If either controller is disconnected the program will need to be restarted in order to reconnect. Sends modified controller input to the MovementMap.

class manualControl.ControllerEvent(name, value)¶

Controller event object. To be sent to the servo handler to convert a button press to movement, for example. Contains a predetermined name and the current event’s value. For buttons the value is 1 or 0. For analog axes it is 0-255. For d-pad hat axes it is -1,0,1.

async manualControl.process_events(device)¶

Async helper function. Process PS4 events and create ControllerEvent objects to be passed to the movementMap
